代码实现了完整的互补带死区SPWM功能：

    1.预计算互补正弦波数据表，主通道和互补通道相位相差180度；
    2.集成死区时间保护机制，防止上下管同时导通；
    3.支持可调节的SPWM频率和死区时间；
    4.使用双PWM通道输出互补信号，适用于电机驱动和逆变器应用。

输出gpio端口需要接低通滤波电路，以过虑掉高频调制信号
低通滤波器说明

RC低通滤波器由电阻和电容组成，允许低频信号通过而衰减高频信号。

截止频率公式: fc = 1/(2πRC)

工作原理: 电容对高频信号的阻抗较小，高频信号被电容旁路到地。

应用场景: 音频处理、信号调理、抗混叠滤波等。
